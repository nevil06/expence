# Expense Manager App Assets

## Required Assets

This app requires the following assets to run properly on mobile:

1. `icon.png` - App icon (1024x1024 recommended)
2. `splash.png` - Splash screen image 
3. `adaptive-icon.png` - Android adaptive icon
4. `favicon.png` - Web favicon (16x16 or 32x32)

## How to Add Assets

1. Create or download appropriately sized images for each asset
2. Place them in this assets folder
3. Ensure they are properly formatted (PNG/JPG)

## Asset Specifications

- App Icon: Square, 1024x1024 pixels, PNG format
- Splash Screen: 1280x800 pixels (or similar aspect ratio), PNG format
- Adaptive Icon: 108x108 pixels, PNG format (with transparent background)
- Favicon: 16x16 or 32x32 pixels, PNG format

## Generating Assets

You can use tools like:
- Figma
- Adobe Photoshop
- Online tools like https://appicon.co/
- Expo's asset generator

## Alternative

You can also use a tool like `expo install expo-image` to help manage placeholder images in your app until you add the proper assets.
